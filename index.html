Calculadora

Este proyecto es la creación de una calculadora simple web con Flet y en el lenguaje de programación Python. La aplicación muestra una interfaz limpia y sencilla con un contenedor que incluye un título personalizado y elementos visuales bien definidos. Es una base perfecta para construir aplicaciones más complejas.

Comenzar Ahora Documentación

¿Qué es Flet?

Flet es un framework moderno para crear aplicaciones de escritorio y web multiplataforma utilizando Python. Permite desarrollar interfaces gráficas (GUI) de forma rápida y sencilla sin necesidad de aprender lenguajes complejos como JavaScript o C++.

🖥️ Multiplataforma

Crea una sola aplicación y ejecuta en Windows, macOS, Linux, iOS, Android y Web.

⚡ Rápido y Eficiente

Desarrollo ágil con hot reload, componentes reutilizables y un SDK optimizado.

🎨 Diseño Moderno

Interfaz limpia, responsiva y con componentes visuales modernos incluidos.

Vista Previa de la Aplicación

La aplicación muestra una interfaz limpia y sencilla con un contenedor que incluye un saludo inicial, un título personalizado y elementos visuales bien definidos. Es una base perfecta para construir aplicaciones más complejas.

Requisitos

  • Python 3.8 o superior
  • Flet 0.84.0 o superior
  • pip (gestor de paquetes de Python)

Instalación

Paso 1: Crear Entorno Virtual

Se recomienda usar un entorno virtual para aislar las dependencias del proyecto:

$ Terminal / CMD
python3 -m venv venv

Paso 2: Activar Entorno Virtual

En macOS/Linux:

$ Bash/Zsh
source venv/bin/activate

En Windows:

$ CMD
venv\Scripts\activate

Paso 3: Instalar Flet

$ Terminal
pip install flet

Ejecución de la Aplicación

Existen tres formas de ejecutar tu aplicación Flet:

Opción 1: Ejecución Directa

La forma más simple de ejecutar la aplicación como escritorio nativo:

$ Terminal
flet run

Opción 2: Ejecución Específica

Ejecutar un archivo específico como aplicación de escritorio:

$ Terminal
flet run main.py

Opción 3: Ejecución Web

Ejecutar la aplicación como aplicación web en el navegador:

$ Terminal
flet run --web

Para más detalles sobre las opciones de ejecución, consulta la Guía de Inicio Rápido de Flet.

Estructura del Proyecto

La organización actual del proyecto Flet con estructura modular:

Estructura de carpetas
fletApp/
├── .gitignore # Archivos ignorados por Git
├── pyproject.toml # Configuración del proyecto Python
├── index.html # Página web (este archivo)
├── README.md # Documentación completa
└── src/ # Código fuente de la aplicación
├── main.py # 🐍 Punto de entrada principal de la app Flet
└── assets/ # Recursos de la aplicación
└── pantallaApp.png

📁 Raíz del Proyecto

Archivos de configuración, documentación y página web. Los archivos .gitignore y pyproject.toml están en la raíz para una mejor organización.

📦 Carpeta src/

Contiene todo el código fuente de la aplicación. El archivo main.py es el punto de entrada único de tu aplicación Flet.

🖼️ Carpeta assets/

Recursos estáticos como imágenes, iconos y otros archivos multimedia utilizados por la aplicación.

Compilación para Diferentes Plataformas

Flet permite compilar tu aplicación para múltiples plataformas. Usa el comando flet build seguido del nombre de la plataforma y la bandera -v para ver más detalles.

Plataformas Disponibles

📱 Android flet build apk -v

Crea paquetes APK y AAB

🍎 iOS flet build ipa -v

Genera archivos IPA

🍎 macOS flet build macos -v

Crea aplicación macOS

🐧 Linux flet build linux -v

Compila para Linux

🪟 Windows flet build windows -v

Ejecutable para Windows

🌐 Web flet build web -v

Aplicación web
